# boards
react、lumen、mysql、nginxを利用したdocker上の掲示板サイト<br><br>

[react]<br>
docker exec -it react /bin/bash<br>
yarn install<br>
npm run build<br><br>

[lumen]<br>
docker exec -it lumen /bin/bash<br>
composer install<br>
envの設定をする。データベースとgoogleアカウント<br>
（googleアカウントで信頼性の低いサービスを許可しないといけない。）<br><br>
